SweetAlertDark
-----------------
>基于SweetAlert的深色弹窗组件https://www.sweetalert.cn/

>调用方法:
><script type="text/javascript">
>swal("标题","内容","图标类型");
></script>

>图标类型
>错误/失败：error
>成功/正确：success
>信息/提示：info
>警告/重要：warning

>重要提示

1. SweetAlertDark不可与SweetAlert同时加载

2. 请务必设置网页编码utf-8否则会出现中文乱码！
可以在head加入charset=UTF-8"
